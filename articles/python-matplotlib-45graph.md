### 2. コーディング

右斜め 45 度に伸びる直線のグラフを描きます

```python
import matplotlib.pyplot as plt

# グラフとして描画するデータ
# x軸とy軸の値を配列で設定する
x = [1,2,3,4]
y = [2,3,4,5]

# x軸とy軸の値を引数にプロットします
plt.plot(x, y)
plt.show()
```
