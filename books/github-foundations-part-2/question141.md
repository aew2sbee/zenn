## Q141: GitHub のフロープロセスにおける「コミットメッセージ」の意義とは？

### 選択肢

- 変更を自動的にメインブランチにマージします。
- リポジトリのバックアップを作成します。
- プロジェクトの履歴を追跡するために、コミットで行われた変更の概要を提供します。
