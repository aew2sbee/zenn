## Q119: GitHub の組織レベルのリポジトリで利用できるロールは、次のうちどれですか？(5 つ選んでください)

### 選択肢

- コミットをプッシュする
- issues とプルリクエストを管理する
- リポジトリの削除
- リポジトリの設定を変更する
- 新規リポジトリの作成
