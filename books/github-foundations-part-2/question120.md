## Q120: 組織内の GitHub Actions に関するセキュリティマネージャーの能力はどれですか？

### 選択肢

- 組織全体で許可またはブロックされるアクションを設定できる。
- 90 日以上経過したプライベートリポジトリのすべてのワークフローを削除できる。
- 組織内のリポジトリのコードレビューをオフにすることができます。
- ブランチ保護ルールの「マージ前にステータスチェックをパスする必要がある」を削除することができます。
