## Q124: GitHub ディスカッションで実行できるアクションはどれですか？(三つ選んでください)

### 選択肢

- ディスカッションをピン留めする
- 他のリポジトリのディスカッションに転送する
- ディスカッションを秘密の gist に変換する
- issue に変換する
- 既存のプルリクエストにマージする
