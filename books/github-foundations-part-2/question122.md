## Q122: GitHub Copilot for Business が提供し、GitHub Copilot for Individual が提供しない機能はどれですか？

### 選択肢

- 公開コードに基づくモデル提案の制限を含む、組織全体のポリシー管理
- 有料利用料の免除
- 24 時間 365 日の GitHub Copilot チームによる電話サポート
- GitHub Copilot の背後にある AI モデルのローカルホスティング
