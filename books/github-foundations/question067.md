## Q67: git を説明するのに最も適切なものはどれですか？

### 選択肢

- Git は分散型バージョン管理システムだ。
- Git は集中型バージョン管理システムです。
- Git はバージョン管理を可能にする GitHub の製品である。
- Git はファイル共有システムです。
