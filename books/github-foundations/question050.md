## Q50: 個人の GitHub アカウントで作成したリポジトリで、友人と一緒に作業するにはどうすればいいでしょうか？

### 選択肢

- リポジトリの設定で、彼をコラボレーターとして招待する。
- `CODEOWNERS` ファイルに追加してください。
- 個人リポジトリでは共同作業はできません。組織所有のリポジトリでのみ可能です。
- 友人に GitHub の認証情報を提供する
