## Q3: GitHub のプロジェクト、issues、プルリクエスト間の同期はどのように行われるのですか？

### 選択肢

- issues やプルリクエストの更新は、自動的に GitHub Projects に反映されます。この統合は双方向に機能し、プロジェクト内のプルリクエストや issues の情報を変更すると、プルリクエストや issues にもその情報が反映されます。
- issues やプルリクエストの更新は自動的に GitHub Projects に反映されません。
- 情報の同期は、プロジェクトから issues やプルリクエストへの一方向にしか機能しません。issues やプルリクエストの更新は、GitHub Project のアイテムには自動的に反映されません。
- 情報の同期は一方向にしか機能しません - issues やプルリクエストからプロジェクトへ。プロジェクトアイテムへの更新は、issues やプルリクエストには自動的に反映されません。
