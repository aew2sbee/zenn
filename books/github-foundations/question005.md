## Q5: GitHub Project の掲示板に課題やプルリクエストを追加するには、どのような方法がありますか？

### 選択肢

- 個別に、自動で、または一括で追加できる。
- 手動で 1 件ずつのみ追加できる。
- 個別または一括追加のみ可能。
- プロジェクトのワークフローを使用して自動的にのみ行う
