# Q5: GitHub プロジェクトボードに Issue や Pull Request を追加する方法には、どのような方法がありますか？

## 選択肢

- 個別に、自動で、または一括で追加できる。
- 手動で1件ずつのみ追加できる。
- 個別または一括追加のみ可能。
- プロジェクトワークフローを使った自動追加のみ可能。
