## Q70: GitHub のリポジトリとは何ですか？

### 選択肢

- コードやファイル、各ファイルのリビジョン履歴を保存する場所です。
- Docker イメージや NPM パッケージを保存する場所です。
- ブラウザ上でソースコードを編集できるビジュアルコードエディタです。
- GitHub のリポジトリは、開発者がコード関連の問題を議論できるチャットルームです。
