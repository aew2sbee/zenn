## Q30: GitHub Codespace の環境を停止した場合、まだコミットしていない変更は後で戻ってきて確認できますか？

### 選択肢

- はい。GitHub Codespace を停止して再起動したときのデフォルトの動作です。
- いいえ。変更は失われます。停止する前にコミットする必要があります。
