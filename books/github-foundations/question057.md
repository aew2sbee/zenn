## Q57: GitHub でユーザーをフォローする効果は？

### 選択肢

- そのユーザーの公開活動について、あなたの個人ダッシュボードに通知が届きます。
- そのユーザーの個人ダッシュボードに、あなたの公開活動に関する通知が届きます。
- フォローリクエストが承認されると、そのユーザーのプライベートリポジトリにアクセスできるようになります。
- GitHub チャットを使って、そのユーザーとプライベートな会話をすることができます。
