## Q98: GitHub Wiki とは何ですか？

### 選択肢

- Wikis はドキュメントをホスティングするためのリポジトリ機能です。
- Wikis は GitHub の公式ドキュメントです。
- Wikis はコードスニペットをホストする機能です。
- Wikis はプロジェクトの`README.md`ファイルを保持する場所です。
