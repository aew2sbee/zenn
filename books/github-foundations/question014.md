## Q14: もしあなたが、Issue や Pull Request に対して同じコメントを頻繁に書く場合、時間を節約するためにどの GitHub 機能を使うべきですか？

### 選択肢

- 保存済み返信（Saved replies）
- コメントテンプレート（Comment templates）
- リポジトリテンプレート（Repository templates）
- ラベル（Labels）
