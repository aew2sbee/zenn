## Q55: GitHub スポンサープログラムとは何ですか？

### 選択肢

- オープンソースプロジェクトの開発者を財政的に支援する方法です。
- GitHubに資金を提供する企業グループです。
- 技術業界への就職を支援するプログラムです。
- GitHubの使い方を学ぶ学生を、高いスキルを持った開発者がボランティアでサポートするプログラムです。
