## Q86: GitHub Desktop とは？

### 選択肢

- コンピュータ上で Git や GitHub を操作するための GUI アプリケーションです。
- 自分のサーバーやパソコンにインストールできる GitHub のセルフホスティング版です。
- GitHub の個人アカウント向けの料金プランで、無料プランに加えて追加機能を提供します。
- ブラウザ上でリポジトリにアクセスできるオンラインエディタです。
