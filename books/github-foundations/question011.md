## Q11: ラベルの使用用途（ユースケース）にはどのようなものがありますか？

### 選択肢

- Issue や Pull Request の分類に使う。
- リポジトリ内のファイルを分類するために使う。
- リリースに割り当てて、リリースノートに含めるために使う。
- リポジトリのコントリビューターにラベルを割り当てて、役割や権限を示すために使う。
