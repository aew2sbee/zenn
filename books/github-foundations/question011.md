## Q11: ラベルの使用例とは？

### 選択肢

- Issue や Pull Request を分類する
- リポジトリ内のファイルを分類する
- リリースにラベルを割り当て、リリースノートに含める
- リポジトリの貢献者にラベルを割り当て、プロジェクトでの役割や権限を示す
