## Q88: プルリクエストを課題にリンクする方法は？(2 つ選んでください)

### 選択肢

- iOS と Android 用のモバイルアプリで、携帯電話から GitHub で作業することができます。
- プルリクエストの説明でキーワードを使用し、課題番号を参照する。
- プルリクエストの説明にキーワードを使用し、課題のタイトルを参照する。
- コードの変更に課題番号を含める
- コミットメッセージに課題番号を含める
