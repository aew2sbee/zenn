## Q13: issues やプルリクエストに担当者を割り当てるにはどうすればよいですか？

### 選択肢

- サイドバーの Assignees フィールドを使用する。
- その issues やプルリクエストに GitHub ユーザー名のラベルを追加する。
- issues やプルリクエストの説明文に記載する
- issues やプルリクエストに人を割り当てることはできません。
