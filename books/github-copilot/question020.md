## Q20: GitHub Copilot を使って開発者のワークフローを強化することの利点は、次のうちどれでしょう？

### 選択肢

- 開発者の生産性を高めるために、コード・スニペットを提案することができる。
- すべてのプロジェクトにおいて、コードレビューの必要性を完全に代替します。
- 人間の承認なしにプルリクエストを自動的にマージします。
- 単一のプログラミング言語で書かれたソフトウェアにのみ対応します。
